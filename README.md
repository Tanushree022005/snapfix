## SnapFix: AI Code Debugger
An AI-powered tool built with **Google Gemini 1.5 Flash** that helps students debug code from a simple photo.

##  Features
* **OCR Code Extraction:** Reads code from screenshots.
* **Instant Fixes:** Identifies bugs and provides corrected code.
* **Friendly Explanations:** Explains the "why" behind the error.

## Built With
* [Streamlit](https://streamlit.io/)
* [Google Gemini API](https://aistudio.google.com/)
